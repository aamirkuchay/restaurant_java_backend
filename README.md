# Restaurant Backend API

This project is a Spring Boot-based backend application for managing restaurant operations.

## Features

- Menu management
- Order processing
- Table reservations
- Customer management


## Technologies Used

- Java 17
- Spring Boot 3.1.0
- Spring Data JPA
- MySQL
- Maven

## Getting Started

### Prerequisites

- JDK 17 or later
- Maven 3.6+
- PostgreSQL 13+
